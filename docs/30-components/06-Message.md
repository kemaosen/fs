# Message
### 基础用法
从顶部出现，3 秒后自动消失。
<ClientOnly>
  <Message-Default></Message-Default>
</ClientOnly>
### 不同状态 && 不同位置
<ClientOnly>
  <Message-Status></Message-Status>
</ClientOnly>

### 可关闭
可以添加关闭按钮。
<ClientOnly>
  <Message-Close></Message-Close>
</ClientOnly>
### 使用 HTML 片段
`message` 属性支持传入 HTML 片段
<ClientOnly>
  <Message-Html></Message-Html>
</ClientOnly>

### Attributes
<ClientOnly>
  <Message-Attributes></Message-Attributes>
</ClientOnly>
